# Support CODI

CODI is an open source AI software architect and coding agent built to help developers design, build, and ship better software.

If CODI has saved you time, helped solve a difficult problem, or simply made development more enjoyable, consider supporting the project.

Your contributions help fund:

* Continued feature development
* AI research and experimentation
* Infrastructure and testing
* Documentation and tutorials
* Bug fixes and performance improvements
* Long-term maintenance of the project

CODI will remain open source. Contributions are completely optional, but every donation helps keep development moving forward.

Thank you for helping build the future of developer tools.
